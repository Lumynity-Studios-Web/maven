\## Maven AutoGenerate



1. \*\*AutoGenerate - Simple\*\*

It is a one-file json loaded by GenMavenArtifacts.jar to automatically generate a maven artifact from data provided.
If something is wrong, the app will throw an error and stop.



2\. \*\*AutoGenerate - Complex\*\*

A one-file json loaded by GenMavenArtifacts.jar to automatically generate multiple maven artifacts for multiple submavens, projects (artifact\_id), and files (numbered in order of processing)
